# 🛒 E-Commerce Website
![E-Commerce Banner](https://via.placeholder.com/1000x300)

A modern, responsive **E-Commerce Web Application** built using HTML, CSS, and JavaScript. This project provides a smooth shopping experience with product listing, cart management, and clean UI design.

---

## ✨ Features

- 🛍️ Product listing with images and prices  
- 🔍 Search and filter products  
- 🛒 Add to cart / remove from cart  
- 💰 Cart total calculation  
- 📱 Fully responsive design (mobile + desktop)  
- ⚡ Fast and lightweight UI  
- 🎨 Clean and modern interface  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- LocalStorage (for cart data persistence)

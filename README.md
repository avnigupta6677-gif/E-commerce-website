# 🌟 Overview

A modern and responsive **E-Commerce Website** that provides users with a seamless online shopping experience.

### Key Features

- 🛍️ Browse Products
- 🔍 Search Products
- 🛒 Add to Cart
- 💰 Automatic Price Calculation
- 📱 Fully Responsive Design
- ⚡ Fast & Lightweight
- 🎨 Modern User Interface
